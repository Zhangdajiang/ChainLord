## 📋 创建步骤

### 第一步：访问创建页面
https://programmablesearchengine.google.com/controlpanel/create

### 第二步：填写基本信息
- **搜索引擎名称**：`加密货币与全球新闻搜索`
- **搜索语言**：选择"中文"和"英文"

### 第三步：添加网站
根据上述方案选择，复制粘贴网站列表到"输入网站或网页"框

### 第四步：获取凭证
1. 创建完成后，记录 **搜索引擎 ID**
2. 访问 https://console.cloud.google.com/apis/credentials
3. 创建 **API 密钥**
4. 启用 **Custom Search API**

### 第五步：更新 .env 文件
```env
GOOGLE_API_KEY=你的API密钥
GOOGLE_SEARCH_ENGINE_ID=你的搜索引擎ID
```

## 🔧 API 使用限制

### Google Custom Search API
- **免费额度**：100 次查询/天
- **付费价格**：$5 / 1000 次查询（超出免费额度后）
- **最大结果数**：每次查询最多 10 条结果

### Tavily API（对比）
- **免费额度**：1000 次查询/月
- **优势**：专为 AI 优化，结果更适合分析
- **推荐**：先用 Tavily，Google Search 作为备用
